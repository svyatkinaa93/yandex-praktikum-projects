# Data-Analytics-project.

Данный репозиторий содержит несколько проектов, которые я делал в рамках обучения Анализу Данных в Яндекс.Практикуме.

В рамках курса я научился:
* работать в python-библиотеках: pandas, numpy, SciPy, scikit-learn, matplotlib, seaborn;
* делать предобработку, предварительный и исследовательский анализ данных;
* выявлять скрытые закономерности в данных;
* проводить расчёт бизнесовых и маркетинговых метрик;
* проверять гипотезы и оценивать результаты А/Б теста методами математической статистики;
* создавать дашборды с метриками на базе Tableau;
* строить прогнозы и проводить кластеризацию методами машинного обучения.

## Проект 1: Аналитика в Яндес.Афише.
На основе данных о посещениях сайта Яндекс.Афиши я изучил, как люди пользуются продуктом, 
когда они начинают покупать, сколько денег приносит каждый клиент, когда он окупается. 
Расчитал и визуализировал метрики пользовательской активности, Retention Rate, Churn Rate, LTV, CAC, ROMI.
Дал рекомендации бизнесу на основе исследования.

## Проект 2: Проверка гипотез по увеличению выручки в интернет-магазине. Оценка результатов А/В теста.
Приоритезировал гипотезы. Провёл оценку качества подготовки тестовой и контрольной групп. 
Построил и проанализировал графики кумулятивной конверсии и среднего чека по группам;
Расчитал статистическую значимость различий в метриках на сырых и очещенных данных;

## Проект 3: Проверка гипотез по увеличению выручки в интернет-магазине. Оценка результатов А/В теста.
Построил модель предсказания оттока клиентов методами машинного обучения. 
Проанализировал метрики разных моделей и выбирал лучшую.
Провёл кластеризацю клиентов методом K-means и сформировал портрет клиента каждого кластера.
